# SingleTango
Single Tango is a golang focused chat bot project which uses API.AI as its default conversational engine.

# Getting Started
```bash
git clone https://github.com/SingleTango
export SINGLE_TANGO_KEY=<YOUR API AI CLIENT ACCESS TOKEN>
go run main.go
```
